## About The Project

RO: Un site de tip market unde utilizatorii pot cumpăra/vinde acțiuni, cu preț care fluctuează în timp real. Implementat cu ASP.NET. Pentru materia IS (inginerie softare). 
</br>
EN: A market-type website where users can buy/sell stocks, with prices that fluctuate in real time. Implemented with ASP.NET. For SE (Software Engineering) course. 

### Built With

[![ASP.NET][ASP.NET]][ASP.NET-url]

<!-- USAGE EXAMPLES -->
## Usage

Once signed in, users can view a large number of fictional stocks and buy/sell shares with fictional money. Users can also read fictional financial news !
</br>
Admin users get some more options like adding stocks, news and triggering certain market events.
